## Commandes réservées a certaines utilisatrices

| Commande            | Message                                                                                                                             |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| !goto @utilisateur  | met un timeout à utilisateur de 15 secondes                                                                                         |
| !groto @utilisateur | met un gros timeout à utilisateur de 1 heure                                                                                        |
